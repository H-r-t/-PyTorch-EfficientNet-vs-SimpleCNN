# flower_data-PyTorch-EfficientNet-SimpleCNN
基于pytorch的花卉图像识别项目，采用两类核心算法：深度学习模型（CNN + 迁移学习）；传统机器学习模型（SVM），从零训练卷积神经网络（SimpleCNN）SimpleCNN 为自主设计的浅层卷积神经网络结构，主要由：多层卷积 + Batch Normalization + ReLU，最大池化层（Max Pooling），全局平均池化（AdaptiveAvgPool），全连接分类层。
代码：“花卉图像识别.ipynb"，数据集："flower_data"
数据集来源：kaggle:  https://www.kaggle.com/datasets/sunrise100/flower-data
